Lab2: การหาข้อมูล tag

  1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร ?
    2. ตอบ มีหน้าที่กำหนดประเภทเอกสาร ในที่นี่คือ html
  2. ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด ?
    3. ตอบ ใช้ Attribute <title>
  3. แท็ก <b>,<i> ต่างจาก <strong>,<em> หรือไม่อย่างไร ?
    4. ตอบ ถ้าเป็น <b> หรือ <i> จะเป็นการเน้นคำในส่วนที่มองห็นเท่านั้น ไม่ได้ให้ความสำคัญในการ search engine แต่ถ้าเป็น <strong> และ <em> จะเป็นการให้ความสำคัญกับตัวนั้นๆด้วย
  4. หา tag ที่สามารถแสดงขอความกลับหลังได้
    5. ตอบ tag : <bdo> สามารถทำได้
