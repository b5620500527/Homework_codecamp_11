1. เลขที่ถูก alert เป็นลำดับสุดท้ายคืออะไร
2. Code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่
    let i = 0;
    while (++i < 5 ) alert(i);
    
    let i = 0;
    while (i++ < 5 ) alert(i);

3. code ทั้งสองอันนี้จะแสดง alert ออกมาเหมือนกันทั้งหมดหรือไม่
    for (let i = 0; i < 5; i++) alert(i);

    for (let i = 0; i < 5; ++i) alert(i);
